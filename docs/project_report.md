# Project Report: Risk Analytics Dashboard for Australian Banking Sector

## Objective
Develop a data-driven decision-support system that improves lending decisions using client profiling, portfolio segmentation, and risk scoring.

## Problem Statement
How can Australian banking teams reduce lending risk while improving the speed and consistency of applicant screening?

## Solution
This project builds a modern four-page risk dashboard using Australian state-level banking data, a MySQL-style dimensional model, and Python-based risk scoring.

## Outcome
The dashboard allows analysts to monitor lending exposure, deposits, default rates, high-risk clients, credit score distribution, and state-level lending risk.
